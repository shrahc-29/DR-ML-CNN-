**Diabetic Retinopathy Detection and Classification using Deep Learning**

This repository contains deep learning models developed for the automated detection and classification of Diabetic Retinopathy (DR) from retinal fundus images. The project focuses on both binary classification and multi-class grading of DR severity using Convolutional Neural Networks (CNNs) and transfer learning techniques.


**Project Overview**

Diabetic Retinopathy is a diabetes-related eye disease that can lead to vision loss if not detected early. This project aims to assist in early diagnosis through AI-based image analysis using state-of-the-art deep learning architectures.


**Features**
- Binary classification of DR and Non-DR retinal images
- Multi-class classification of DR severity stages
- Training from scratch and transfer learning approaches
- Models trained on publicly available retinal fundus datasets
- Data preprocessing and augmentation pipeline
- Performance evaluation using accuracy, precision, recall, F1-score, confusion matrix, and ROC-AUC
- GPU-compatible training using Google Colab/Kaggle


**Models Used**
- ResNet50
- VGG19
- DenseNet121
- EfficientNetB0


**Technologies**
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- OpenCV
- Matplotlib
- Scikit-learn
- Dataset


**The models were trained on retinal fundus image datasets such as the DDR dataset and other publicly available DR datasets.**


**Objectives**
- Improve early-stage DR detection
